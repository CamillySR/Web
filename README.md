# 🌐 Camilly Ramos — Web Portfolio

Personal portfolio showcasing my HTML and CSS practice projects, built as part of my journey learning web development.

## 🔗 Live Preview

> Enable **GitHub Pages** in repository Settings → Pages → Deploy from branch (main) to view this online.

## 📁 Project Structure

```
Website/
├── index.html          ← Portfolio homepage
├── login.html          ← Login page with form design
├── index3.html         ← Product page with navigation
├── forms.html          ← HTML form elements practice
├── audios.html         ← HTML5 audio player
├── tables.html         ← Styled HTML table
├── index4.html         ← CSS box model practice
├── video.html          ← HTML5 video element
├── style.css           ← Shared styles
├── login.css           ← Login page styles
└── images/             ← Image assets
```

## 🛠️ Technologies

- **HTML5** — semantic structure, forms, media elements
- **CSS3** — Flexbox, box model, float, custom styles
- **Git & GitHub** — version control

## ▶️ Running Locally

```bash
git clone https://github.com/CamillySR/Website.git
cd Website
```

Open `index.html` in your browser. For live reload, use the **Live Server** extension in VS Code.
